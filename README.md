Un mini-site de type Linktree, créé pour centraliser tous mes liens importants (réseaux sociaux, projets, contacts, etc.) dans une interface moderne et responsive.
Développé en HTML, CSS et JavaScript, avec un design épuré et adaptable à tous les écrans.

Fonctionnalités :

Liens personnalisés avec icônes et couleurs

Mise en page responsive (mobile & desktop)

Animation légère au survol

Facilement modifiable (code clair et organisé)
